SketchVault စာရွက်စာတမ်း

ဖိုင်တင်ခြင်း နှင့် Supabase

  1. src/config.js ထဲတွင် Supabase URL နှင့် anon key ထည့်ပါ။
  2. supabase/schema.sql ကို Supabase SQL Editor တွင် run လုပ်ပါ။
  3. Authentication → GitHub (နှင့် Email) ကို Enable လုပ်ပါ — Google မလိုပါ။
  4. Redirect URLs တွင် production domain နှင့် #/auth path ပါဝင်ရန် သေချာပါစေ။
  5. ဖိုင်တင်ခြင်းသည် browser မှ Supabase Storage သို့ တိုက်ရိုက်ဖြစ်သည် — hosting (Vercel/Cloudflare) သည် static ဖိုင်များကိုသာ ပေးပါသည်။
  6. Android တွင် တင်မရပါက app ထဲ upload status နှင့် browser console တွင် [SketchVault Upload] log များကို ကြည့်ပါ။
